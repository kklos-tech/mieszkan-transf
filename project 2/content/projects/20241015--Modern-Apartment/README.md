# Project Images

To add images to this project, place JPG or PNG files in this folder with the following naming convention:

- `001-Title.jpg` - Main hero image and thumbnail
- `002-Title.jpg` - Additional project image
- `003-Title.jpg` - Additional project image
- etc.

Example filenames:
- `001-Living-Room-View.jpg`
- `002-Kitchen-Design.jpg`
- `003-Bedroom-Layout.jpg`

The first image (001) will automatically be used as the project thumbnail and hero image in the portfolio gallery.
